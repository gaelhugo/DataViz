```bash
pip install virtualenv
```

2. Create a virtual environment

```bash
virtualenv venv
```

3. Activate the virtual environment

```bash
source venv/bin/activate
```

4. Install the dependencies

```bash
pip install -r requirements.txt
```

5. Run the server

```bash
npm run server
```

6. Run the client

```bash
npm run build
```
